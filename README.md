## About Proffy
Proffy is a study project developed during Rocketseat's NLW. This repository contains the mobile application of said project.

<br /> <br />

## Features

- Axios
- Expo
- React Native
- TypeScript

## Skills/Concepts Learned

- API consumption using axios
- AsyncStore
- Hooks
- Navigation
- Prop Types

## Getting started

Import the codes

```sh
yarn start
```
